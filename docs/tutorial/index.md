## Learning Path

**Recommended progression:**

1. Start with [Loading & Visualizing](use_dod2k.md) to understand the database structure
2. Explore [Loading & Merging](load_merge.md) to work with source databases
3. Learn [Duplicate Detection](duplicate.md) for data quality control
4. Follow [Recreate from Scratch](from_scratch.md) for the complete pipeline

---

## Prerequisites

Before starting these tutorials, ensure you have:

- ✓ Python 3.8 or higher
- ✓ Jupyter notebook environment
- ✓ DoD2k repository cloned locally
- ✓ Required dependencies installed

See the [Installation Guide](../getting_started/installation.md) for setup instructions.
